# Nuxt Frontend Project

基于 Nuxt 3 的前端项目。

## 开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 目录结构

```
frontend/
├── assets/          # 静态资源（CSS、图片等）
├── components/      # Vue 组件
├── pages/           # 页面组件（路由）
├── public/          # 公共静态文件
├── app.vue          # 应用根组件
├── nuxt.config.ts   # Nuxt 配置文件
└── package.json     # 项目依赖
```

详细文档请参考 [Nuxt 官方文档](https://nuxt.com/docs)。
